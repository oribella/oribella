### Development

After cloning the mono repo

### Installation

Install dependencies:

```sh
yarn
```

### Lint

```sh
yarn run lint
```

### Test

Run the tests:

```sh
yarn test
```

### Debugging

For easy debugging using [vscode](https://code.visualstudio.com/download)

Add a breakpoint in a `spec.ts` and hit F5 and off you go...

### Build

To generate umd and es format:

```sh
yarn run build
```
